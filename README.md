## Notes of Javascript Course

### Syntax

; character after each command
```js
command... ;
command... ;
command... ;
```
{} to create code hierarchies
```js
{ // top-level scope
  { // nested scope
  }
}
```
() to prioritize operations
```js
var x = 2;
var y = 3;
var z = (x + y) * 3; // z = 15
```
